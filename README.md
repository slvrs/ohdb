# Key-Value Storage

Организация хранилища ключ-значение для использования в OpenHAB

Состоит из следующих скриптов:
- "ohdb_put [имя ключа] [значение]" записывает в базу значение
- "ohdb_get [имя ключа]" выводит в STDOUT значение из базы. При отсутствии
данных выводит пустую строку
- "ohdb_del [имя ключа]" удаляет ключ из базы
